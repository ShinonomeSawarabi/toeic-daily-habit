# TOEIC 段階的習慣トラッカー v2

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GitHub Pages](https://img.shields.io/badge/demo-live-brightgreen)](https://[your-username].github.io/toeic-habit-tracker-v2/)

TOEIC学習を段階的にレベルアップさせていく習慣トラッキングアプリ。王冠バッジ、リセット機能、X（Twitter）投稿機能を搭載。

![App Screenshot](https://via.placeholder.com/800x600?text=TOEIC+Habit+Tracker+v2)

## ✨ v2の新機能

### 👑 王冠バッジシステム
- レベル完了時に自動的に王冠を獲得
- ホーム画面に「獲得した王冠」セクションを表示
- 完了日時と共に王冠の履歴を保存
- レベル一覧で完了したレベルに👑アイコン表示

### 🔄 リセット機能
- **個別レベルリセット**: 特定のレベルの進捗と王冠のみを削除
- **全データリセット**: 全レベルの進捗、王冠、履歴を一括削除
- 確認ダイアログで誤操作を防止

### 🐦 X（Twitter）投稿機能
- タスク完了後にXで成果を報告
- 投稿内容をプレビュー確認
- 日付、レベル、進捗、残り日数を自動生成
- ハッシュタグ自動追加（#TOEIC習慣トラッカー）

## 🎯 基本機能

### レベルシステム（8段階）
1. **Level 1**: 基礎固め期（英単語10個 × 10日）
2. **Level 2**: 文法ドリル期（文法ドリル5問 × 10日）
3. **Level 3**: 単語力強化期（英単語30個 × 14日）
4. **Level 4**: でる1000 開始期（50問 × 4日）
5. **Level 5**: でる1000 継続期（4日解いて1日復習 × 20日）
6. **Level 6**: Part 7 強化期（長文10問 × 15日）
7. **Level 7**: 総合演習期（模試1セット × 7日）
8. **Level 8**: 弱点克服期（苦手パート50問 × 10日）

### カスタマイズ可能
- 各レベルの内容を自由に編集
- タスク内容、必要日数を変更可能
- 新しいレベルの追加
- デフォルト設定に戻す機能

### 進捗管理
- 今日のタスクをワンタップで完了記録
- レベルごとの進捗バー表示
- 必要日数達成で自動レベルアップ
- 月間カレンダーで履歴を視覚化

### 統計機能
- 総学習日数
- 現在の連続日数（ストリーク）
- 最長連続日数
- 月間達成率

2. `index.html`をブラウザで開く

それだけ！サーバーのセットアップは不要です。

## 🎮 操作方法

### ホーム画面
- **今日のタスク**: 緑のボタンをタップして完了を記録
- **X投稿**: タスク完了後、青いボタンでXに投稿
- **王冠セクション**: 獲得した王冠を確認
- **カレンダー**: 今月の学習履歴を確認
  - 紫色: 完了した日
  - 赤色: 未完了の過去日
  - 青枠: 今日
- **統計カード**: 学習データをひと目で確認

### レベル画面
- 全レベルの進捗状況を一覧表示
- 完了レベルには👑アイコン
- 現在のレベル、完了したレベルがわかる

### 設定画面
- 各レベルをタップして編集
- 「🔄 リセット」で個別レベルをリセット
- 「新しいレベルを追加」で独自のレベル追加
- 「🗑️ 全データをリセット」で一括リセット（警告セクション）

### X投稿機能
1. タスクを完了
2. 「🐦 Xで成果を報告する」ボタンをタップ
3. プレビュー画面で内容を確認
4. 「🐦 Xに投稿」でX投稿画面が開く
5. 内容が自動入力されているので「ポスト」するだけ

**投稿例:**
```
📚 TOEIC学習記録
2025年2月15日

Level 1: 基礎固め期
3日目完了！
あと7日でレベルアップ 🎯

#TOEIC習慣トラッカー #TOEIC学習 #継続は力なり
```

## 💾 データ保存

- すべてのデータはブラウザのローカルストレージに保存
- インターネット接続不要（オフラインでも使えます）
- データはデバイスに保存されるため、プライバシーも安心

**注意**: ブラウザのキャッシュをクリアするとデータが消えます

## 📖 カスタマイズ例

### 自分だけのレベルを作る

設定画面から以下のような独自レベルを作成できます：

```
レベル名: Level 9
タイトル: リスニング特訓期
説明: シャドーイングで耳を鍛える
タスク: Part 3, 4 を各10問シャドーイング
必要日数: 12
```

### 学習スタイルに合わせる

- **短期集中型**: 各レベルの必要日数を短く設定
- **じっくり型**: 必要日数を長めに設定
- **試験直前型**: Level 7, 8 だけを使う
